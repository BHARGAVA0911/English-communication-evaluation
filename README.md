# English Communication Skill Evaluation using AI

This project evaluates spoken English communication using Natural Language Processing and Speech Analysis.

The system evaluates five key dimensions:

1. Grammatical Accuracy
2. Fluency
3. Pronunciation
4. Vocabulary Richness
5. Coherence and Cohesion

## Input Data

The system takes two inputs:

1. Audio recording of speech
2. Transcript generated using Whisper
